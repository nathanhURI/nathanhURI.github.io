CSC 492 Semester-Long Project

Milestone 1

1. Problem. Describe the problem you chose and how the web-application will help users.


2. Target users. Characterize the user population (VERY IMPORTANT). Eventually you will be asked to show your web app to people from this population and ask for feedback. So think carefully here. Remember, you may have multiple user classes (and not just one).


3. Solution. Describe one possible solution to the problem --- i.e., the interface that you envision, and how it will address the problem. You are not committed to your solution, since you may find after building and evaluating some prototypes that a wholly different solution will work better. That's perfectly fine.
